## Prerequisites

- Node.js (v18.x or higher)
- npm (v9.x or higher)
- Git
- Supabase account
- Google Cloud account
- Modern web browser

## Getting Started
- Make sure to create an account on supabase
- Create google cloud service project
- Install ngrok to host the server locally
